Portfolio

Aerobatic Sports Plane

This was a project done over the course of the Fall 2022 semester during the ME408 (Aircraft Performance & Design) class. The goal of the project was to design an aerobatic sports plane capable of meeting a list of customer requirements within the time-frame of the class.

Boston Curb Flooding

This is a problem statement exercise for the class ME360 (Product Design). The focus was finding a solution for the flooding on Boston's sidewalk crossings.

Pierce School Bridge

This is a problem statement exercise for the class ME360 (Product Design). The focus was finding a solution to the problem seen in the current design of the Pierce School bridge.

Big Belly Assignment

This was a problem statement assignment for Big Belly. Wherein we had to design a solution to the trash-compactor issue when snow settles on the top of the compactor during winter.

Bell Assignment

This assignment was part of the ME360 Product Design course. The objective was to use a 2D sketch to build a 3D model of a bell. Then, using FEA we were to determine the sound that the bell would make.

Double Wedge Assignment

This assignment focused on analysing a double wedge airfoil using compressible flow and fluid dynamics. The analysis was conducted using isentropic state values.

Truss Design Project

This project was part of the EK301 class - wherein we had to design a truss capable of handling a specified load at a given point. A MATLAB program was written to help in determining the max load, the various forces felt by each member, as well as the overall cost of the design.
