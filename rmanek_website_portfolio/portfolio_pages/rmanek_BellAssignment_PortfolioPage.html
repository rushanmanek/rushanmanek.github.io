Bell Assignment

Summary

This assignment focused on recreating a CAD model of a bell, based on a sketch of its cross sectional profile.


From there, a brass material was applied, and FEA frequency analysis was conducted on the model. From the analysis multiple frequencies were obtained. These are frequencies that would occur for the given bell given various forms of deformation (i.e. it being hit and ringing).


These frequencies were then input into a script in MATLAB, and adjusted based on the following equation seen on the right, where:

  • A = Amplitude of frequency
  • alpha = Dissipation Factor
  • omega = Frequency
  • t = Time

Multiple iterations of the code were run in order to fine tune a point where the audio gave the impression of a bell - which can be heard below.
