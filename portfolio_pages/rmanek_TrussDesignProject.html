Truss Design Project

Summary

This project was part of the EK301 class - wherein we had to design a truss capable of handling a specified load at a given point.


In order to help the group achieve the final design, I wrote a program using MATLAB that would help in determining the final output of our design. Including the max load, the various forces felt by each member, as well the overall cost of the design.


The graph on the right is the plotted output of the practice scenario, wherein each member is highlighted as to whether it is undergoing: compression, tension, or no force. As well as highlighting which is the weakest member (i.e. the first one that will buckle).
