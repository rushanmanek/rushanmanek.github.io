Double Wedge Assignment

Summary

This assignment focused on analysing a double wedge airfoil using compressible flow and fluid dynamics. The analysis was conducted using isentropic state values.


The goal of the assignment was to take in a list of inputs (seen below), and calculate the lift coefficient, and drag coefficient of the airfoil.
  • Initial Static Pressure
  • Initial Static Temperature
  • Angle of Attack
  • Forward Half Angle
  • Aft Half Angle
